# V_eff 민감도 분석 (2026-03-30)

## 배경

V_eff 방식 확정(2026-03-30): sDBD 70mm×70mm×1mm → V_eff=4.9cm³.
V_reactor=250cm³ 기존 방식은 폐기 (참고 기록으로만 보존).

**의문**: Dilution factor(V_eff/V_reactor 비율)는 PFR에서 수학적 동치로 무효과인데,
V_eff 자체를 변경하면(250→4.9cm³) 전환율이 달라진다(4.95→4.35%, 303K). 왜?

## 결론 요약

V_eff 민감도의 원인은 **비선형(2차) radical 반응의 불완전 스케일링**.

- 1차 반응(EI, 여기종): Rate×τ ∝ (P_dep)×τ = P/Q = SEI → V_eff 무관 (ratio ≈ 1.0x)
- 2차 radical 반응: radical 밀도가 ne(∝1/V_eff)에 비례하지 않음
  - radical 자기소비(OH+OH, HO₂+HO₂ 등)가 고밀도에서 강화
  - 결과: CH₄에 도달하는 OH/O 감소 → 전환율 하락

## 스케일링 이론

α = V_reactor/V_eff = 250/4.9 = 51.0

V_eff 변경 시:
- P_dep = P/V_eff → α배 증가
- τ = V_eff/Q → 1/α배 감소
- SEI = P/Q → 불변

### 1차 반응 (rate = k × n_e × n_feed)
- n_e ∝ P_dep ∝ α, n_feed ≈ 일정
- Rate ∝ α, Rate×τ ∝ α×(1/α) = 1 → **V_eff 무관**

### Radical 매개 반응 (rate = k × n_radical × n_feed)
- n_radical의 스케일링은 손실 메커니즘에 의존:
  - 1차 손실(벽, 안정분자 반응): n_rad ∝ (생성)/(k_loss) ∝ α → ratio = 1.0
  - **2차 손실(radical+radical)**: n_rad ∝ √(생성/k₂) ∝ √α → ratio = √α/α = 1/√α ≈ 0.14
  - 혼합: 1/√α < ratio < 1.0
- CH₄ 전환 기여: Rate×τ ∝ n_rad/α → 2차 손실 지배 시 감소

## 핵심 연쇄 구조와 스케일링 전파

```
EI (e + O₂/CO₂) → O          [∝ α]
                    ↓
             O + O₂ → O₃      [∝ α, 303K 지배]
             O + HO₂ → OH     [∝ α × √α → 2차 소비]
                    ↓
            O₃ + H → OH       [∝ 1 × H_ratio]
            HO₂ + O → OH      [∝ √α × α_O]
                    ↓
           OH + CH₄ → CH₃     [CH₄ 전환 핵심 경로]
                    ↓
           CH₃ → CH₃O → CH₂O → HCO → CO + HO₂  [radical 재순환]
                                         ↑
                                    O₂ + HCO
```

스케일링 전파:
1. ne ∝ α (전력 밀도 직접 비례)
2. O ∝ α (EI 생성, 1차 손실)
3. HO₂ ∝ √α (radical 생성, **2차 자기소멸**)
4. OH ∝ 14x (HO₂×O에서 생성 → 혼합 스케일링, 1차+2차 소멸 혼합)
5. H ∝ 24x (EI+N₂* 직접 + OH 간접 생성, 1차 소멸)

CH₄ 전환 경로별:
- EI 직접: Rate×τ ∝ α/α = 1 (무감)
- N₂* 직접: Rate×τ ∝ α/α = 1 (무감)
- OH 경로: Rate×τ ∝ 14.1/51 = **0.28** (303K 지배, Δ=−1.20%p)
- O 경로: Rate×τ ∝ 47.6/51 = **0.93** (303K 무감) → **523K에서 0.57** (Δ=−1.34%p)

## 함의

1. **V_eff가 물리적으로 올바르더라도**, radical 비선형 스케일링으로 실험 대비 과소예측.
2. 실험과의 차이(RMSE ~2.7%p)는 모델 한계가 아니라 **0D 모델에서 V_eff 사용 시의 내재적 특성**.
3. 실제 sDBD에서는 discharge zone 외부의 radical 확산/혼합이 있어 0D 모델의 "완전 혼합 discharge zone" 가정보다 radical 밀도가 낮을 수 있음 → 실제로는 비선형 효과가 약화되어 전환율이 더 높을 수 있음.
4. 이 분석은 V_eff sweep에서 RMSE 단조감소(V_eff↑→전환율↑→RMSE↓)의 원인을 정량적으로 설명함.
